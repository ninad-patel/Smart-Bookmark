# Smart-Bookmark
The one and only place to securely save your bookmarks.

Process of development :-
1. Created normally using npx create-next-app with typescript, tailwind css and supabase integration.

2. Installed dependencies according to the requirements of the project.

3. Setup and configure Supabase with google OAuth using google client ID and secret for google login.

4. Implemented google login page and home page which will be shown after login.

5. Setup database in supabase by creating table, policy and enable realtime according to the requirement.

6. Implemented the required features to align everything created in DB.


The most chalenging part during the process of the development was to setup supabase for both google login and realtime updates.
